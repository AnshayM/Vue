Ajax

  Vue.js 并没有限制使用哪种方式进行 ajax 访问，所以可以使用如下方式
     1. 原生 ajax
     2. JQuery
     3. vue-resource
     4. fetch.js
     5. axios.js
   那么到底用哪种方式呢？
     1. 原生一般不在项目中直接用
     2. jquery 不如4，5方便
     3. vue-resource 已经不再维护了，所以不推荐
     4. fetch.js 是 vue.js 官方推荐
     5. axios.js 是vue.js 官方推荐
